    Snake Game

    This is a Snake Game implemented in C using the SDL2 library. The game offers two modes: single-player and two-players. In the single-player mode, the player controls a snake to eat apples and grow longer without hitting itself. In the two-players mode, two players compete against each other, each controlling their own snake, and the game ends when the time runs out.

    FireZone

    This is an online multiplayer battle game developed in C using the SDL2 library. Players connect through a peer-to-peer architecture to engage in fast-paced combat, controlling unique characters with customizable abilities. The game features real-time synchronization, smooth animations, and dynamic battle arenas. Players can strategize, compete, and adapt as they navigate through power-ups, obstacles, and challenging opponents. The game emphasizes precision and quick reflexes, offering an exhilarating skills for competitive players.
